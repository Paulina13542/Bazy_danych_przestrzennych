## Struktura projektu
Wszystkie pliki do ćwiczeń znajdują się w osobnych gałęziach.
Poniżej przedstawiono poszczególne gałęzie oraz odpowiadające im tematy ćwiczeń.
- `cw1.sql` – tworzenie schematu, tabel i danych przykładowych oraz komentarze
- `cw1.1.sql` – zapytania SQL
- `cw2.sql`- praca z danymi przestrzennymi
- `cw3.sql`- Karlsruhe
- `cw4.pdf` - praca w QGIS
- `cw5.sql` -  Praca z kolekcjami geometrii i EWKT
- `cw6-7.sql` -  ćwiczenia z PostGIS raster(tutorial)
- `cw8.sql` - ćwiczenia z PostGIS raster
